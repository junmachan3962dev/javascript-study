### 1. 演算子とは「評価」を促す記号

演算子は、値と値を組み合わせて**新しい一つの値**を作るための記号です。

### 2. 演算子の優先順位

■ **優先順位の基本ルール**

演算子には「強さ」が決まっています。強いものから順に計算され、同じ強さなら左から右へ進みます。

■ **演算子の優先順位**

演算子は数十種類ありますが、実務で「順番」を意識すべきなのは以下のグループだけです。

1. **グループ化**: `( )`
2. **算術演算子**: `*`, `/` → `+`, `-`
3. **比較演算子**: `>`, `===` など
4. **論理演算子**: `&&` → `||`
5. **三項演算子**: `? :`
6. **代入**: `=`

＊優先順位は1.が一番強くて、6.が一番弱い

### 3. 論理演算子と「短絡評価」

- モダンなコードで非常によく使われる、三項演算子よりもさらに短い分岐テクニックです。
- 論理演算子は**値を返す**演算子です。
#### ■ **`&&（AND）`演算子**

最初の `Falsy` を返す。なければ最後の値を返す。

**用途：条件が通った時だけ何かをしたい時**
```
// ログインしている時だけ、メッセージを表示する
const isLoggedIn = true;
const message = isLoggedIn && "おかえりなさい！";
console.log(message); // "おかえりなさい！"
```

#### ■`||（OR)`演算子

最初の `Truthy` を返す。なければ最後の値を返す。

**用途：デフォルト値（予備）の設定**
```
// userの名前があればそれを、なければ "ゲスト" を代入
const name = "" || "ゲスト"; 
console.log(name); // "ゲスト"
```
値が入力されていなくても何かしらの結果を表示したい時に使用

＊**短絡評価（ショートサーキット）とは？**
   論理演算子が「評価を途中で切り上げて、その時点で値を返す」仕組みのことです。



### 4. 比較演算子

「**Boolean（真偽値）**」、つまり **`true`** か **`false`** のどちらか一方だけを返す演算子です。

■ **等価演算子（等しさを比べる）**

| **演算子**   | **意味**     | **補足**                  |
| --------- | ---------- | ----------------------- |
| **`===`** | 厳密に等しい     | **鉄則。** 値も型も同じなら `true` |
| **`!==`** | 厳密に等しくない   | `===` の逆。違うなら `true`    |
| `==`      | 等しい（ゆるい）   | **非推奨。** 勝手に型変換される      |
| `!=`      | 等しくない（ゆるい） | **非推奨。**                |
|           |            |                         |

**=== 演算子のサンプルコード**
```
// 数値の比較
console.log(10 === 10); // true
console.log(10 === 5);  // false

// 文字列の比較
console.log("JS" === "JS"); // true

// 型が違う場合（重要）
console.log(10 === "10"); // false
```

**!== 演算子のサンプルコード**
```
// 値が「違う」ことを確認する
console.log(10 !== 5);    // true  (10と5は違うので「正解」)
console.log(10 !== 10);   // false (同じなので「不正解」)

// 型が違えば、値が同じに見えても true（違う）になる
console.log(10 !== "10"); // true  (数値と文字列は「違う」ので正解)
```
■ **関係演算子（大小を比べる）**

| **演算子**  | **意味** | **補足**              |
| -------- | ------ | ------------------- |
| **`>`**  | より大きい  | 左が右より大きければ `true`   |
| **`<`**  | より小さい  | 左が右より小さければ `true`   |
| **`>=`** | 以上     | 左が右と同じか大きければ `true` |
| **`<=`** | 以下     | 左が右と同じか小さければ `true` |

```
// 大小の比較
console.log(10 > 5);  // true
console.log(10 < 5);  // false

// 「以上・以下」（その数を含む）
console.log(10 >= 10); // true
console.log(10 <= 9);  // false
```


### 5. Truthy（真）と Falsy（偽）について

JavaScriptの**すべての値**は、条件式や論理演算の中で「真」か「偽」として扱われる性質を持っています。

■ `Falsy（false）` な値（これだけ覚えれば OK！）

以下の **7つだけ** が `false` 扱いになります。
- `false`
- `0`（数値のゼロ）
- `-0`
- `""`（空文字）
- `null`
- `undefined`
- `NaN`（Not a Number：計算不能）

■ `Truthy（true）` な値

**上記以外すべて** です。特に間違いやすい「真（true）」扱いになるものは以下の通りです。
- `" "`（スペース 1 つでも入っている文字列）
- `"0"`（文字列のゼロ）
- `[]`（空の配列）
- `{}`（空のオブジェクト）

＊**どちらの値も演算子の判定に重要な判定基準です。**

＊それぞれの種類の演算子は、**条件式や論理演算の時には`Falsy`, `Truthy`で判定され、論理演算では値、比較演算子では、真偽値を返す**。
